Text clock


April 1, 2017

Tecnologies: Arduino  c++  electronics 

On my desk at work I have this cool text clock.

Important notes:

RTC and LCD does not play nicely, so the code may seem a little long. That’s because I had to use U8G2_XXXX::drawStr which take const char as argument.

Also the output is in norwegian. The code itself has enough comments where appropriate so that you can switch to your own language. Also for those who understand norwegian I had trouble finding a font that support norwegian characters; i.e. å.

Components:

  • Picture frame
  • Arduino Uno
  • LCD
  • RTC
  • Switch
  • Wires

Wiring:
